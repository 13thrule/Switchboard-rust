Enterprise Features

⚡ Phase 4: Local IPC

100x faster message passing for same-host connections via shared memory. 2µs latency vs 200µs TCP.

🔄 Phase 5: Wildcard Patterns

Lock-free trie router with O(depth) matching. Supports * (single-level) and > (recursive) patterns.

🎯 Zero-Copy Broadcasting

All subscribers read the same memory location. Adding 1,000 subscribers costs essentially zero extra memory.

🔐 Lock-Free Architecture

Uses crossbeam SkipMap and DashMap for concurrent access without mutex contention on the hot path.

⏰ Waker-Driven Event Loop

Never polls. 0% CPU at idle. Subscribers wake instantly when messages arrive using Tokio channels.

🌐 WebSocket Support

Browser clients use the same binary protocol as TCP. Demo runs directly in your browser.
