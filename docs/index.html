🚀 Enterprise Features

Purpose-built for high-performance systems requiring sub-microsecond latency and zero memory overhead.

Phase 4: Local IPC

100x faster message passing for same-host connections via shared memory. Atomic head/tail pointers, lock-free ring buffer, 2µs latency.

Phase 5: Wildcard Patterns

Lock-free trie router with O(depth) matching. Supports * (single-level) and > (recursive) wildcards.

Zero-Copy Broadcasting

All subscribers read the same memory location. Adding 1,000 subscribers costs essentially zero extra memory or CPU.

Lock-Free Architecture

Uses crossbeam SkipMap and DashMap for concurrent access. No mutex contention on the hot path.

Waker-Driven Event Loop

Never polls. 0% idle CPU. Subscribers wake instantly when messages arrive via Tokio channels.

WebSocket Support

Browser clients use the same binary protocol as TCP. Try the interactive demo below with live protocol inspection.
